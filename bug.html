Reporting a bug

First,check that the issue you're facing is not already part of the TODO list

ios-driver is currently under development. Bugs are fixed and feature added quickly. When you file a bug, make sure you add the version you're using. You can find the version hitting the url http://localhost:4444/wd/hub/status . Copy the build part, that looks like :


"build": {
    "revision": "b0e9250cafe9870fc3b0a0224cb6e8d242aa1374",
    "time": "20130129-1201",
    "version": "0.5.0-SNAPSHOT"
},
"ios": {"simulatorVersion": "6.1"},
"java": {"version": "1.6.0_37"},
"os": {
    "arch": "x86_64",
    "name": "Mac OS X",
    "version": "10.8.1"
}

and file a bug on github
