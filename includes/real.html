Preparing the application.

You need to compile and package the app using your developer provisioning profile.This is a requirement from Apple's UIAutomation. . Ideally, you should get the ipa file with the proper provisioning provile from your CI server.If you don't, you can follow the following steps to generate what you need from XCode.

Start XCode and load the code of the application you want to test. Select the application in the left panel, and choose Product > Archive.

That should automatically open Xcode's archive panel. Select the archive you just created and choose Distribute.

This will trigger a prompt to choose what kind of distribution you want. For testing, it is "Save for entreprise or Ad Hoc Deployment".Click Next

Choose a dev provisioning profile of type Development ( You need an IOS dev account to get this profile ).

Click export. The generated ipa file is what you need to provide when ios-driver starts.
